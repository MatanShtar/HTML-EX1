# 🐕 HTML-EX1: Much Web. Very Design. 🐈

Welcome to my very first HTML & CSS home assignment! 

This repository is a highly professional, enterprise-grade demonstration of my web development skills, my grasp of folder hierarchies, and my undeniable love for internet culture.

## 🚀 Features

* **Page 1: The Doge Dimension:** Contains over 10 paragraphs of high-quality developer Doge lore. Complete with functional anchor links, smooth scrolling, and a custom warm-purple theme. 
* **Page 2: The Feline Files:** A carefully curated, ordered list of the internet's best cats (featuring the legendary Soldier Cat and Exploding Cat).
* **Responsive Styling:** Includes custom CSS to ensure maximum *wow* and minimum *bork*.

## 📂 Project Structure

* `/Assets` - Where the good boys and cats live (PNGs and JPEGs).
* `/css` - The magic stylesheets that make the pages look pretty.
* `index.html` - The main Doge homepage.
* `index2.html` - The dedicated Cat zone.
